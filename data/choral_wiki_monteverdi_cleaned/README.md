# Claudio Monteverdi Madrigals Dataset

## Source and License
This dataset is composed of files retrieved from [ChoralWiki (Choral Public Domain Library - CPDL)](https://www.cpdl.org/wiki/index.php/Claudio_Monteverdi). 
Content on CPDL is generally available under their specific copyright licenses (often CPDL, Personal, or Creative Commons, unless otherwise specified). Please refer to the [CPDL Copyright page](https://www.cpdl.org/wiki/index.php/ChoralWiki:Copyright) and the individual metadata of each score for specific licensing details.

## Data Organization
The cleaned dataset is organized into folders for each of Monteverdi's 9 books of madrigals. Each madrigal has its own subfolder, following this naming convention:
`book_number-madrigal_number-madrigal_title`

Inside each specific madrigal folder, you will find exactly three files representing the full score:
* `.pdf` (sheet music)
* `.mxl` / `.xml` (MusicXML for digital notation software)
* `.mid` / `.midi` (audio playback)

## Missing Data
The following madrigals are currently missing specific file formats in the cleaned dataset:
* 6-02: missing `.mxl`
* 6-03: missing `.mxl`
* 6-04: missing `.mxl`

## Dataset Notes and Known Inconsistencies
* **Transcription Variations:** The notation in the scores is not entirely consistent across the dataset. The transcriptions were created by various authors and editors, meaning engraving styles, editorial decisions, and formatting will vary.
* **Voices and Instruments:** Across the different books, there are natural changes in the number of voice parts and instrumentation (e.g., transition from a cappella to basso continuo and concerted works).
* **Mismatched Files:** In some cases, the `.mid` file may have been generated by a different editor or from a different edition than the corresponding `.pdf` and `.mxl` files. 
* **Numbering:** The numbering of the madrigals was retrieved directly from the ordered lists on the ChoralWiki pages. If you spot any indexing errors, please report them.

## Troubleshooting and The "Uncleaned" Folder
If you encounter issues with a specific file (especially out-of-sync or broken `.mid` files), please check the original `choral_wiki_monteverdi` (uncleaned) folder. 

The uncleaned folder contains the raw scrape and includes:
* Alternative transcription versions by different authors.
* Isolated PDF parts (e.g., separate basso continuo parts, single voice lines, or instrument parts).
* Additional file formats.
